About This mini project

  • Sign In (Login) :

    • Users can log in with their username and password
    • Authenticates users using Django’s built-in authentication system.
    • Shows error messages on invalid credentials.

  • Sign Out (Logout) :

    • Users can securely log out.
    • Displays informational message on logout.

  • Sign Up (Registration) :

    • Checks for existing usernames to avoid duplicates
    • Creates new users and provides confirmation messages.

  • Role-Based Access :

    • The upload file button is restricted to superusers.
    • File deletion allows only the uploader or a superuser to delete.
    • Only authenticated user can download files

  • File Listing :

    • Supports filtering by department and year.

  • File Deletion :

    • Supports deleting files with permission checks.
    • Deletes both the file and database record.

  • Technologies Used :

    • Frontend :
    • HTML
    • CSS
    • Backend :
    • Django
